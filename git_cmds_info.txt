git add q1.sh --->> Add files to staging area

git commit -m "initial commit message" --->> Commiting of files (creates a local copy of the files specified on your system (local repo))

git reset -->> to remove files from staging area that were added by mistake (to be used after (git add) and before (git commit)

git branch demo_branch -->> creates a new branch

git checkout demo_branch -->> shifts to the specified branch

git branch -d demo_branch -->> delets the branch
